## 📝 项目简介

本项目旨在打破原版 HOI4 相对单一的法律选择机制，通过引入多层级的 **Branch (分支)** - **ID (插槽)** - **Value (法案值)** 结构，实现更具深度和策略性的国家管理体验。

目前主要聚焦于 `branch_1: 国防事务与人力资源` 的重构，涵盖了从征兵制度、服役年龄到参军福利等维度。
